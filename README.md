## How It Works

Every day at 8:00 AM PST, an automated pipeline:

1. **Fetches** all papers from [HuggingFace Daily Papers](https://huggingface.co/papers)
2. **Analyzes** each paper in parallel â€” extracting key innovations, methods, and takeaways
3. **Generates** per-paper markdown summaries with figures and reproduction guides
4. **Builds** a beautiful sci-fi themed daily webpage
5. **Publishes** to GitHub Pages automatically

### Setup

Live mode needs X API v2 bearer token:

```bash
export X_BEARER_TOKEN="<your_x_api_v2_bearer_token>"
```

### Run (mock mode, no token required)

```bash
python3 scripts/fetch-x-snapshot.py --mode mock --date 2026-02-12
python3 scripts/build-site.py
```

Or end-to-end:

```bash
X_MODE=mock ISSUE_DATE=2026-02-12 ./scripts/run-daily-pipeline.sh
```

### Run (live mode)

```bash
python3 scripts/fetch-x-snapshot.py --mode live --date 2026-02-12
python3 scripts/build-site.py
```

Or end-to-end:

```bash
X_MODE=live ISSUE_DATE=2026-02-12 ./scripts/run-daily-pipeline.sh
```
